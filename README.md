# Name Trends Analysis in the United States

A BigQuery and SQL analysis project that studies U.S. baby-name trends from 1910 to 2020, including regional diversity, gender-neutral naming patterns, decade-level popularity shifts, and cultural influence.

## Files Included

- `138_Project.ipynb` - Jupyter notebook with SQL queries and visualizations
- `final_report.docx` - technical report
- `Presentation.pptx` - project review presentation
- `ER_Diagram.png` - dataset entity relationship diagram

## Project Features

- Tracks the most popular names by decade using window functions.
- Identifies gender-neutral names with balanced usage across genders.
- Ranks states by name diversity.
- Detects fastest-rising names after 1990.
- Measures cultural naming impact through public data.
- Finds names that remain stable across decades.

## Engineering Value

- Shows SQL analysis, public dataset exploration, query optimization, and communication of analytical findings.
- Useful foundation for analytics engineering, data storytelling, and product-insight reporting workflows.
